# R-type

Jogo desenvolvido através ambiente Allegro para um trabalho prático da disciplina de Programação de Software 1, depois do desenvolvimento do jogo foi feita a apresentação com o objetivo de mostrar as grandes áreas que um engenheiro de sistemas agregar. Ainda fazendo melhorias no jogo em relação a desempenho gráficos.
